# Minegumo Otoschedule

音MAD周りの生放送配信の情報を集めた「音MAD周辺配信表」の情報を自動で通知するbot、『音MAD周辺配信通知bot』です。

## 対応サービス

- Bluesky
- Misskey
- Discord
